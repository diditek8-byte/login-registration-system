# login-registration-system

A full-stack user authentication system featuring login and registration, built with PHP and MariaDB. 

Features
- User login
- User registration
- User logout
- Session-based authentication

Technologies
- HTML
- CSS
- JavaScript
- PHP
- MariaDB
- Apache
- Ubuntu Linux

Installation
1. Transfer the project files to the Ubuntu server.
2. Install and configure Apache, PHP, and MariaDB.
3. Create the required database and tables.
4. Update the database credentials in config.php.
5. Place the project in the Apache web root
6. Open the website on browser with IP adress

What I learned
- Building a login and registration system
- Working with Databases
- Using PHP sessions for user authentication
- Deploying a PHP application on Ubuntu Linux
- Configuring an Apache web server
